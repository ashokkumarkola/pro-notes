# Figma shortcuts & tips

### 1. Tools – Basic

| Action           | Windows Shortcut | Mac Shortcut     |
| ---------------- | ---------------- | ---------------- |
| Move / Select    | V                | V                |
| Scale            | K                | K                |
| Hand (Pan)       | H                | H                |
| Pen              | P                | P                |
| Frame            | F                | F                |
| Text             | T                | T                |
| Rectangle        | R                | R                |
| Ellipse          | O                | O                |
| Line             | L                | L                |
| Slice            | S                | S                |
| Create Component | Ctrl + Alt + K   | Cmd + Option + K |
| Place Image      | Shift + Ctrl + K | Shift + Cmd + K  |

### 2. Auto Layout

| ---------------------------------- | ------------------------------------------------ |

| Remove Auto Layout                 | Shift + Alt + A (Win) / Shift + Option + A (Mac) |

| Action                    | Shortcut                          |
| ------------------------- | --------------------------------- |
